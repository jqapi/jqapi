.dequeue()

    Execute the next function on the queue for the matched elements.

    .dequeue(queueName)🡢 jQuery

    queueName String A string containing the name of the queue. Defaults to fx, the standard effects queue.

    When .dequeue() is called, the next function on the queue is removed from the queue, and then executed. This function should in turn (directly or indirectly) cause .dequeue() to be called, so that the sequence can continue.

    <button>Start</button>
<div></div>

    div {
  margin: 3px;
  width: 50px;
  position: absolute;
  height: 50px;
  left: 10px;
  top: 30px;
  background-color: yellow;
}
div.red {
  background-color: red;
}

    $("button").click(function () {
  $("div")
    .animate({ left: "+=200px" }, 2000)
    .animate({ top: "0px" }, 600)
    .queue(function () {
      $(this).toggleClass("red").dequeue();
    })
    .animate({ left: "10px", top: "30px" }, 700);
});
